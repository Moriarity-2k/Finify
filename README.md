## Getting Started
 To run ComicStash locally, follow these steps:
  1. Clone the repository:    


```bash 
git clone https://github.com/Moriarity-2k/Finify/
```


2. Open the folder in your terminal

``` bash
cd Finify 
 ```
 
 3.  install the dependencies
 ```bash 
 npm i && npm run dev
  ```
